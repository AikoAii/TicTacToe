# Tic-Tac-Toe

```txt
c language | minimax algorithm | cli game | recursion ( •̀ ω •́ )✧
```

A sophisticated, **unbeatable Tic-Tac-Toe AI** built with pure C and the Minimax algorithm.
No brute force, no hardcoded moves — just recursive game theory and clean logic.

Modular, professional, and impossible to defeat.

---

## ⊹₊ Features

```txt
[✓] Unbeatable AI (Minimax)

[✓] Zero dependencies (Pure C)
```

All decisions are computed dynamically using a recursive search tree.

```bash
gcc (GNU Compiler Collection)

```

---

## ⌯⌲ Getting Started

Clone the repository:

```bash
git clone https://github.com/AikoAii/TicTacToe.git
cd TicTacToe
```

Build the project:

```bash
make
```

Run the game:

```bash
./tictactoe
```

---

## ⚡︎ Usage

The game starts with the AI making the first move.

Enter a number **1–9** to place your move:

```txt
 1 | 2 | 3
-----------
 4 | 5 | 6
-----------
 7 | 8 | 9
```

* `X` = Human
* `O` = AI

The AI will instantly respond with the optimal move.
